# LiffeyCare UAT Checklist

- Verify patient can register, login, and reach dashboard.
- Verify doctor account requires admin approval before access.
- Verify appointment booking blocks double-booking for same doctor/slot.
- Verify reschedule and cancellation update patient and doctor dashboards.
- Verify reminder triggers for booking confirmation and cancellation.
- Verify admin can review logs and analytics widgets.
- Verify responsive views in Chrome/Firefox desktop and mobile viewport.
